# Highlighted Text and Assistance Remarks

---

## Highlights

### "Comment" Template

"**Claim:**", "**Lemma:**", "**Theorem:**", and "**WTS:**" are all highlighted under this template, as well as any text, if any, that follows on the same line. "**Proof:**" and "**Pf:**" are also highlighted, but not the text that follows them as proof bodies are usually long and would ideally start a line below.

### "Keyword" Template

The keywords "**DFA**", "**NFA**", "**GNFA**", "**RL**", "**NRL**", "**CFL**", and "**NCFL**" are all highlighted under this template, as they are abbreviations of concepts that frequently come up in this class.

### "String" Template

All strings of language {0, 1} in the state transition format we came up with in class are highlighted under this template, except with inner <> for individual states in place of inner brackets for better readability.

### "Invalid" Template

All binary strings of 0b[01]* format that are multiples of 5 are highlighted under this template.
